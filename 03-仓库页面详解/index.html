03 · 仓库页面详解

这篇的目标：学会看仓库页面——认识所有标签栏、了解右侧面板信息、看懂文件浏览区、创建你的第一个仓库。

1. 什么是仓库页面

当你进入一个仓库（如 github.com/facebook/react），看到的就是仓库页面。它是 GitHub 最核心的页面类型，所有代码、Issue、PR、讨论都在这里管理。掌握它的布局就等于掌握了 GitHub 80% 的操作。

2. 仓库顶部信息区

2.1 仓库名和路径

页面左上角的面包屑导航显示：组织/用户名 → 仓库名，如 facebook/react。

旁边标注仓库类型：

  • Public（公开）—— 所有人都能看到
  • Private（私有）—— 只有你和协作者能看到

2.2 仓库统计按钮（右侧）

位于仓库名称右侧的按钮行：

按钮作用
Watch设置通知：Participating / All / Ignore
Fork复制仓库到你的账号
Star收藏/点赞，Star 数是项目流行度的重要指标
Sponsor赞助该项目（如果项目开启了赞助）
Add to list将仓库归类到你的 Star lists

每个按钮旁边显示数字（Watch 人数、Fork 数、Star 数）。

3. 仓库标签栏（Tabs）

仓库页面中部有一行标签栏，是切换功能区的核心导航：

标签说明
Code代码浏览、文件管理、README（默认视图）
Issues问题追踪、Bug 报告、功能请求
Pull Requests代码审查与合并
ActionsCI/CD 自动化工作流
Projects项目看板管理
More（更多）展开下拉菜单

More 菜单包含：

子页面说明
Wiki维基文档
Security安全分析与 Dependabot
Insights数据统计与贡献图
Settings仓库设置（需要管理员权限）

4. 右侧面板（About）

仓库页面的右侧有一个信息面板，显示项目的 About（简介）：

区域内容
Description仓库的简短描述
Website项目官网链接
License开源许可证（如 MIT）
Code of Conduct行为准则
Contributing贡献指南
Security Policy安全策略
Custom properties仓库自定义属性
Branches / Tags / Activity分支数、标签数、活跃度

5. Code 标签页主内容区

Code 标签页是仓库页面默认显示的视图。

5.1 文件浏览器顶部工具栏

元素作用
分支选择器切换分支，默认 main
Tags查看版本标签
Go to file按文件名快速跳转（快捷键 t）
Add file直接在 GitHub 上创建或上传文件
Code 按钮展开：Clone URL（HTTPS / SSH）、下载 ZIP

5.2 最新提交信息

工具栏下方显示最近一次提交的信息：提交者头像+名字、提交信息、提交 SHA（短编号）、时间、commit hash（点击可查看详情）、History（查看提交历史）。

5.3 文件和文件夹列表

以表格形式展示当前目录的文件和文件夹：

列内容
Name文件名/文件夹名
Last commit message最后一次提交的信息
Last commit date最后一次提交的时间
  • 点击文件夹进入子目录
  • 点击文件查看文件内容

5.4 README

文件列表下方是 README.md 的渲染内容。这是仓库的名片，通常包含：项目介绍和用途、安装和使用方法、技术栈说明、贡献方式。

进入一个新仓库，先看 README 是最快的了解方式。

6. 创建你的第一个仓库

方法一：顶部导航创建

点击右上角 ➕ Create new… → New repository。

方法二：Dashboard 创建

在 Dashboard 左侧的 Top Repositories 底部，有 Create new 按钮。

创建步骤

字段说明建议
Repository name仓库名称简短有意义（如 my-first-repo）
Description可选描述一句话说明用途
Public / Private公开或私有初学者选 Public
Initialize with README自动创建 README建议勾选
Add .gitignore忽略规则根据主要语言选择
Choose a license开源许可证建议 MIT

点击 Create repository 完成。

✅ 成功标志：看到新仓库页面，里面有 README 文件。

小结

  • 认识仓库页面的整体布局和标签栏
  • 知道 Watch / Star / Fork 按钮的作用
  • 会用 Code 标签页的文件浏览器
  • 创建了自己的第一个仓库
