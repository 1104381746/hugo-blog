# 入门篇：Codex Desktop App 安装与初次体验

> 面向新手的 Codex Desktop App 系列第一篇。读完本文，你会知道桌面版和命令行版有什么区别、怎么装、装完后第一件事该做什么。

---

## 1 这篇给谁看

你在以下情况，Desktop App 可能比 CLI 更适合你：

| 你的情况 | 推荐 Desktop | 推荐 CLI |
|---------|:---:|:---:|
| 想要图形界面，不想碰终端 | ✅ | ❌ |
| 需要后台自动执行任务 | ✅ | ❌ |
| 同时管理多个项目对话 | ✅ | ⚠️ |
| 只想在终端里快速搞代码 | ❌ | ✅ |
| 服务器/SSH 环境 | ❌ | ✅ |
| 本地开发，喜欢开 IDE 写代码 | ✅ | ✅ 均可 |

Desktop App 和 CLI 共用同一个 AI 引擎，区别在于交互方式和额外能力。

---

## 2 系统要求

| 项目 | 最低要求 |
|------|---------|
| 操作系统 | Windows 10+ / macOS 12+ / Linux（具体看官网说明） |
| 网络 | 需要能访问 openai.com |
| 账号 | OpenAI 账号（免费注册） |

**不需要** 提前装 Node.js、Python 或 Git——Desktop App 自带运行环境。

---

## 3 下载与安装

### 第1步：去官网下载

打开浏览器，访问 [https://codex.chat](https://codex.chat)（或者 OpenAI 官网的 Codex 页面），找到 **Download** 按钮，选对应你操作系统的版本。

✅ **成功标志**：下载得到一个安装包（Windows 是 `.exe` 或 `.msi`，macOS 是 `.dmg`）。

### 第2步：安装

**Windows 用户**：双击安装包 → 可能弹出"Windows 保护了你的电脑"——点击"更多信息"→"仍要运行"→ 按提示走完安装向导。

**macOS 用户**：双击 `.dmg` → 把 Codex 拖进 Applications 文件夹。

✅ **成功标志**：桌面或开始菜单/启动台里出现 Codex 图标。

### 第3步：首次启动与登录

双击图标打开 Codex。

首次启动会弹出登录窗口，你需要用 OpenAI 账号登录（支持 GitHub / Google / 邮箱注册）。如果没有账号，按窗口提示注册一个。

> ⚠️ **风险提示**：登录用的是你的 OpenAI API 额度。Desktop App 使用时会消耗 token，请留意用量，避免意外超支。

✅ **成功标志**：登录完成后看到主界面——左侧是会话列表，右侧是对话区，底部有输入框。

---

## 4 主界面长什么样

登录后你会看到这样一个布局（实际界面以你安装的版本为准）：

大致分几个区域：

| 区域 | 作用 |
|------|------|
| 左侧栏 | 会话列表、设置入口、插件管理 |
| 中间区 | 对话主区域，Codex 的回复都在这里 |
| 底部输入框 | 你打字的地方，支持 `/` 命令 |
| 右下/侧边 | 文件树、终端面板（可切换） |

刚开始不需要记住所有位置，先知道"对话在中间，输入在底下"就够了。

---

## 5 第一个对话

在输入框里打一行字试试：

```
帮我创建一个 hello.txt 文件，内容写上 "Hello, Codex Desktop!"
```

然后回车。

你会看到：
1. Codex 开始思考（显示"思考中..."）
2. 生成回答
3. 如果涉及文件操作，可能会弹出一个 **审批提示**（默认 `on-request` 模式）
4. 点击"允许"或"确认"，Codex 就会执行

✅ **成功标志**：你的电脑上多了一个 `hello.txt`，里面写着 `Hello, Codex Desktop!`。

> 💡 如果没弹审批，说明你当前的审批策略是 `untrusted`（完全信任）。后面会说怎么调整。

---

## 6 理解审批策略（很重要，新手必读）

Codex Desktop 默认的审批策略是 **`on-request`（请求时确认）**，意思是：

- 读文件 → 不需要问你
- **写/改/删文件** → 弹窗问你"可以吗？"
- 执行终端命令 → 也弹窗问你

这是保护你的第一道防线，**建议新手不要关掉它**。

| 策略 | 意思 | 什么时候用 |
|------|------|-----------|
| `on-request` | 写文件/跑命令前问你 | ✅ 新手默认，日常工作 |
| `untrusted` | 全自动，不问直接干 | ⚠️ 你完全信任 Codex 时 |
| `never` | 从不执行 | 🔒 只读分析场景 |

可以在设置里改，也可以对话中打 `/permissions` 随时查看和切换。

> ⚠️ **风险提醒**：切换到 `untrusted` 意味着 Codex 可以自由修改你的文件。除非你很清楚自己在做什么，否则保持 `on-request`。

## 7 常见问题

### Q1：安装包打不开 / 提示不安全

Windows 可能会拦截未签名应用。点击"更多信息"→"仍要运行"。这是正常的，因为新软件尚未获得足够的下载量通过 SmartScreen 自动信任。

### Q2：登录报错 / 连不上

检查网络是否能访问 openai.com。如果在大陆使用，可能需要自行解决网络问题。

### Q3：订阅和费用

Codex Desktop App 本身可以免费下载和使用，但有使用限额。具体定价以 OpenAI 官方公布为准。
